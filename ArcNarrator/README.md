*在 ARC NARRATOR 中，用户可以绘制代表剧情张力变化的弧线作为输入，LLM会根据该弧线的起伏特征生成一段故事，并结合图像生成模型产出的分镜插图，以视觉小说的形式进行多模态展示。*

*运行步骤：*
1. 安装依赖：
npm install
2. 在 .env.local 文件中将 GEMINI_API_KEY 设置为你的 Gemini API 密钥
3. 运行应用：
npm run dev

*In ARC NARRATOR, users draw a curve representing narrative tension as input. Based on the fluctuations of this curve, the LLM generates a corresponding story and integrates storyboard illustrations produced by an image generation model, delivering a multimodal presentation in the format of a visual novel.*

*How to Run：*
1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local] to your Gemini API key
3. Run the app:
   `npm run dev`
